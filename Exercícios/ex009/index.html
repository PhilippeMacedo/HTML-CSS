Trabalhando com Listas

Listas ordenadas (ordered lists) são aquelas em que a ordem dos elemento do conjunto faz diferença (seria o arranjo da matemática). Quando fazemos listas, é comum que entre um elemento no meio e esculhambe a ordem, para isso ser a tag ol e cada elemento pe li. A formatação padrão é feita em números. Assim , quando um novo elemento for inserido, você não precisa renumerar toda a lista. No html5, fechar o li se tornou opcional.

  1. Acodar
  2. Meditar
  3. Tomar café
  4. Escovar os dentes
  5. Deixar o elemento aberto

A Lista tem um parâmetro opcional (o padrão é números) type em que você pode selecionar entre números, letras e algorismos romanos. Há também o parâmetro start, para dizer em qual númeração a Lista é iniciada

Listas não-ordenadas

Usa-se a tag ul (unordered lists), nesse tipo as ordens dos elementos não faz diferença.

  • Fechar a tag li
  • Deixar aberta a tag li

Tem o parâmetro type: disc (padrão), circle e square
