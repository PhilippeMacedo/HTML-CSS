Principais formatações

Negrito / Destaque

Podemos fazer isso de duas maneiras:
1- puramente visual (não semântica) = "<b> </b>.Ex.: Nessa frase, tempos um termo em negrito = bold

2 - semântica = <strong> </strong>Ex.: agora eu usei um negrito semântico = strong
Visualmente, são iguais, mas somente a segunda é semântica!

Itálico/Ênfase

1- puramente visual: Para deixar o termo em itálico, baste usar a tag <i> </i>. Outra maneira, é selecionar com o mouse o trecho e usar o comando ctrl + shift + p , abrir o "envelope com abreviatura" e digitar i e depois dar enter

2- semântica: tag EM (vamos usar aquele atalho novamente)
Alguns comandos quebram a linha, e outros não (comandos expansíveis e de divisão)

Texto marcado

Podemos criar também um texto marcado usando a tag mark

Infelizmente, a tag mark ainda não é automatizada no visual code, nem temos como escolher a cor do marca-texto no html5, isso porque cor não é sigficado (hmtl), é formatação (css)

Quando queremos só fazer uma formatação pontual , de uma coisa específica, podemos usar CSS no meio, com a tag style e selecionar o parâmetro que queremos (depois da tag ): assim . Nesse caso, a seleção específica se sobrepõe à seleção global na parte do head (style)

Todavia, quando fazemos a customização dessa forma, os marks seguintes não herdam os atributos

Para criar um estilo padrão, você tem que criar uma tagstyle ma head do documento

Texto grande e pequeno

Tem certas coisas que simplesmente não são fáceis de entender. A tag big (que deixava o texto maior) está depreciada (obsoleta) pela HTML5, mas a tag small (que deixa o texto menor) segue firme e forte na vida da linguagem.

Texto deletado (com risco)

Serve para dizer que o texto deve ser lido, mas não considerado. Basta usar a tag del (a tag strike está depreciada)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase maior e indicar que ele foi adicionado depois.
